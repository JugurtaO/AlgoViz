### Contributions.

```BASH

    # to get the repo localy
    git clone <repo_url>
    cd <repo_name>

    # fork the repo to your github profil


    # to sync with origin

    git remote add origin <your_forked_repo_url>

    git branch your_local_branch
    git checkout your_local_branch

    # to add your changes
    git add *
```
